# MYDB
**在线体验地址：** http://db.blockcloth.cn

项目概述
Simple SQL Database 是一个简易版本的 MySQL 数据库。项目涵盖了数据存储、事务管理、多版本并发控制（MVCC）和索引构建等核心数据库功能，还模拟实现了日志管理和事务状态查询等高级特性。
- 数据的可靠性和数据恢复
- 两段锁协议（2PL）实现可串行化调度
- MVCC
- 两种事务隔离级别（读提交和可重复读）
- 死锁处理
- 简单的表和字段管理
- 简陋的 SQL 解析（因为懒得写词法分析和自动机，就弄得比较简陋）
- 基于 socket 的 server 和 client
